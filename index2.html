About the cards of FutureMon Fight!

There are 4-5 types of cards


Which are-
Monsters,Spells,Items,Impacts,Flags

1st is MONSTERS!

Monsters have sizes,attacks,defense,and critical and some monsters even have abilities. Monsters can be called out up to 3 spaces but also depending on their size.For example, you have 3 monsters with the size of 1 ,2, 3 and we can only call monster till the sum of the monsters is 4. So if you call out the monsters with size 1 and 3 you cannot call anymore monsters even if there is space left but you can call a 0 size monster. So monsters have sizes 1-4. The second card type is ITEMS!!

Items have abilites,critical and attack but no size and defense.The fighter or player can equip items by paying its equip cost (Go below and press "The Remaining things" to know more about cost) and puting it on the flag(A card type). Some items can even attack if there is a monster in the center position.

The next card type is Flags!
 Flags have no attack, defense, size ,critical or size.They only have the things wriiten that will happen if you use that flag card.
